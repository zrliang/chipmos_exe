Detailed Description

A machine_base object in genetic algorithm.

machine_base_t object is used to record its jobs and the job sequence. The job sequence is formed by linked list. Linked list is convenient to insert job in the sequence. root is the head of the job sequence. tail point on the tail of job sequence.

machine_base_t object isn't only record its jobs but basic information about machine such as machine_no and availiable_time (recover time)

machine_base_t object can be embedded in a container structure. ptr_derived_object maintains the relationship, which are pointed to other container structure if it contains machine_base_t object so that machine_base_t can get more information of container structure.
