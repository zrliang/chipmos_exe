Detailed Description

csv_t type is used to read/write csv_t file.

csv_t object is used to deel with csv_t file I/O. There are several csv_t file have byte order remark. csv_t object is able to handle UTF-8 and UTF-16 byte order remark.

csv_t - Constructor of csv_t object for reading file

This constructor is used to read csv_t file. The mode of opening file is specified in mode. if read is true the constructor will immediately read the data. head is used to specify the csv_t file has header, if head is set true but the head has empty columns, the constructor raise exception. r1 and r2 are used to specify the range of rows of csv_t file. If both are -1, all of data are read into to memory. If r1 isn't -1 but r2 is, rows from r1 to the end are read into memory.

Parameters
filename: the csv_t file name
mode: the mode of opening file
read: read data immediately or not
head: specify if data has header
r1: range from r1 if specify
r2: range to r2 if specify

char  delimiter 
)

parseCsvRow () - parse a row in csv file

This function is able to parse a row, which contains special characters, such as '"' and ',', in an element in row. The special character '"' in an element follows another '"', and the element is quoted. For example, the representation of an element with special character the sheet is A"B. The representation in the csv file is "A""B". For another example, the representation of an element in the sheet is A,B. The representation in csv file is "A,B". parseCsvRow is able to parse these kind of elements to recover the original representation.

Parameters
text: pointer to character type of variable, which points on a row in csv file
delimiter: delimiter in csv file
