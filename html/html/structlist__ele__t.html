Detailed Description

A node of double-linked list.

Each node contains basically 4 pointers. next point to next node of current node. prev point to previous node in linked list. ptr_drived_object is a void* type pointer, which can be used to point to its parent object The operations on list_ele_t object have a structure to maintain the function, the structure is LinkedListElementOpreation. list_operations_t allows user to design their own operations on list_ele_t object.

The list nodes are usually embedded in a container structure. ptr_derived_object is a void* type which can be used to point to the address of container structure that help to manipulate structure.
