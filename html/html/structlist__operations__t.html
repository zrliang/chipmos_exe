Detailed Description

The structure to store all operations of struct list_ele_t. The user can define their own operations.

The list_operations_t is used to link to the functions, which we would like to perform on list element, on host or on device. The library provides flexiblility on changing the function which user would like to perform on list element. The variables of list_operations_t are function pointers.
