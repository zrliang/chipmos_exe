Detailed Description

used to manage all da_station_t objects

da_stations_t object manages all da_station_t objects. The program use the production capacity to predict whether the lot will pass D/A station in 24 hours or not. The classification lot in D/A station is by its recipe(bd_id). If lot is in D/A station, lots will be added in arrived queue and if lot is not in D/A station but the lot has traversed to D/A station, the lot will be added into unarrived queue.
